# digit-recognition

This code allows to create and train a custom neural network to recognize handwritten digits. The training data set is the MNIST database.

### Installation

Download the source code.

### Requirements

You need to have Python 3.8 or a later version installed on your computer.

### User guide

You can call from the command line "python main.py" after modifying the parameters of your neural network in the main.py file.
